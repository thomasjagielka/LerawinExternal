## Installation
1. Clone the repository.
 ```sh
 git clone https://github.com/tomaszjagielka/LerawinExternal.git
 ```
2. Build the project using Visual Studio.

## Usage
1. Launch CS:GO.
2. Launch the tool.
3. Show/hide the tool using `Insert`, exit using `End`.
